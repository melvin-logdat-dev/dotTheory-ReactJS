# Scoreboard Challenge

## Overview
This project extends the original course code by enhancing the scoreboard feature.  
The original implementation only crowned the highest scorer.  
I added ranking positions for **gold, silver, and bronze**, introducing placements to make the scoreboard more engaging and informative.

## Features
- 🥇 Gold crown for 1st place  
- 🥈 Silver crown for 2nd place  
- 🥉 Bronze crown for 3rd place  
- No crowns displayed for zero scores  
- Clean, uncluttered UI with clear ranking logic

## Code Challenge
The challenge was to:
1. Modify the ranking logic to support multiple placements.  
2. Ensure crowns appear only when earned.  
3. Maintain performance and readability in the React components.
